# Flashcards Petrobras

Versão em Markdown para leitura direta no Obsidian.

O arquivo `flashcards-petrobras-base.tsv` continua sendo a versão para importação no Anki. Este arquivo `.md` é a versão visível no Obsidian.

## Como usar

1. Leia a frente do card.
2. Responda mentalmente ou em voz alta.
3. Abra `Ver resposta`.
4. Se errar, marque o assunto no caderno de erros e revise no sábado.
