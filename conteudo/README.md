## Como estudar cada dia

1. Abra o dia correspondente em [[../03-plano-1h-dia]].
2. Abra o módulo de conteúdo indicado abaixo.
3. Leia a explicação com atenção.
4. Faça anotações curtas.
5. Resolva as questões de treino.
6. Transforme erros em flashcards.

## Regra de ouro

Não tente decorar tudo na primeira leitura. Para cada assunto, busque responder:

- O que é?
- Para que serve?
- Quando usar?
- Qual é a diferença para conceitos parecidos?
- Como isso aparece em questão?
