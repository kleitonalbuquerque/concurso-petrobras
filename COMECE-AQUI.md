# Comece aqui

Este projeto usa o Obsidian como ferramenta principal de estudo.

## Por que Obsidian primeiro?

- Funciona com arquivos locais.
- Não exige conta.
- O material já está pronto nesta pasta.
- Você consegue estudar, marcar checklists e criar notas sem depender de internet.

## O que é Obsidian?

Obsidian é um aplicativo de anotações que abre uma pasta do seu computador como um "vault". Cada nota é um arquivo `.md`.

Neste projeto, o vault é esta pasta:

`/storage/projects/concurso-ti/petrobras`

## Primeiro acesso

1. Instale e abra o Obsidian.
2. Clique em `Open folder as vault`.
3. Selecione a pasta `/storage/projects/concurso-ti/petrobras`.
4. Abra a nota `00-dashboard-petrobras`.
5. Comece pelo link `03-plano-1h-dia`.

## Como estudar no dia a dia

1. Abra `00-dashboard-petrobras`.
2. Veja a semana atual.
3. Abra o plano `03-plano-1h-dia`.
4. Escolha o tópico do dia.
5. Estude a teoria/aula.
6. Resolva questões.
7. Marque o checklist no plano.
8. Anote erros e crie flashcards.

## Como marcar um item concluído

No Obsidian, um checklist aparece assim:

```md
- [ ] Tópico pendente
- [x] Tópico concluído
```

Para concluir, troque `[ ]` por `[x]`.

## Como criar uma nota nova

Use este padrão de nome:

`YYYY-MM-DD - assunto estudado`

Exemplo:

`2026-04-26 - Ciclos de vida de software`

Depois copie o conteúdo de:

`templates/sessao-estudo.md`

## Caminho recomendado

1. Use Obsidian por 7 dias.
2. Veja se você gosta do fluxo.
3. Ajuste o dashboard e os checklists conforme sua rotina real de estudo.

- `00-dashboard-petrobras.md`: painel inicial.
